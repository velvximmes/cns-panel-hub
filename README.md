# cns-panel-hub

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库旨在集中存储、管理和发布各类独立的 HTML 页面。这些页面可能用于信息展示、工具界面或演示用途，不特定关联某个域名或网站。仓库本身作为一个静态内容集合，便于用户浏览、引用或二次开发。

## 目录说明

仓库根目录下按类型或用途组织文件夹，每个文件夹内包含对应的 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。

```
cns-panel-hub/
├── panels/          # 主要页面归档目录
│   ├── example1/    # 示例页面1
│   ├── example2/    # 示例页面2
│   └── ...
├── assets/          # 公共资源文件
├── templates/       # 页面模板
└── README.md        # 本文件
```

## 页面归档说明

- 每个独立页面应当放置于单独的子目录中，避免文件冲突。
- 页面应尽量自包含，外部依赖（如字体库、图标库）建议使用 CDN 或本地化引用。
- 归档的页面不针对任何特定域名，使用者可根据需要部署到任意环境。
- 页面内容应当合法合规，不包含恶意代码、跟踪脚本或未经授权的数据收集。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，或改进现有页面。
- 提交前请检查页面是否可正常打开，资源引用路径是否正确。
- 请勿在页面内嵌入针对特定域名的硬编码链接或营销内容。
- 仓库维护者会定期审核归档内容，清理无效或过时的页面。

## 使用方式

你可以通过以下方式使用本仓库中的页面：

1. **直接浏览**：克隆仓库后，在本地用浏览器打开 HTML 文件。
2. **静态部署**：将整个仓库或某个页面目录部署到任何静态托管服务。
3. **引用嵌入**：将页面作为 iframe 或组件嵌入到其他项目中（需遵守许可协议）。
